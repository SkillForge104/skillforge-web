# Skill Forge Website

Official website and certificate verification portal for **Skill Forge**.

Skill Forge is a student-driven learning initiative at **Thamirabharani Engineering College**, built around the idea of _learning by doing_ and peer-to-peer knowledge sharing.

This repository hosts the static frontend website used to:

- Present Skill Forge as an initiative
- Document what we do
- Verify the authenticity of certificates issued for short-term learning programs

---

## Features

- Public-facing informational website
- Certificate verification using unique Certificate IDs
- Clear display of participant details and track focus
- Fully static, frontend-only setup
- Deployed using GitHub Pages

---

## Certificate Verification

Each certificate issued by Skill Forge contains a unique Certificate ID.

The verification page allows anyone to:

- Enter a Certificate ID
- View participant name, track, duration, institution, and course focus
- Confirm the authenticity of the certificate

This system is intended for transparency and verification purposes only.
